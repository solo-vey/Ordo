# Reports

This directory holds validation, delivery, release, benchmark, and milestone
evidence. Generated current reports and preserved historical reports have
different roles.

- [`delivery/README.md`](delivery/README.md) — current delivery-gate output.
- [`self-check/README.md`](self-check/README.md) — current package self-check output.
- [`pre-release/README.md`](pre-release/README.md) — pre-release evidence and legacy records.
- [`apf/README.md`](apf/README.md) — relocated legacy APF evidence index.

For current release identity, use [`../manifests/README.md`](../manifests/README.md).
For accepted empirical evidence, use [`../empirical_evidence/README.md`](../empirical_evidence/README.md).
