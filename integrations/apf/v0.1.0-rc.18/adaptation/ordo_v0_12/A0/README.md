# APF A0 — Ordo v0.12 Adaptation Scope and Compatibility Matrix

Status: `ready-for-human-review`

## Purpose

This A0 milestone defines the exact adaptation boundary between:

- APF baseline: `v0.1.0-rc.12-confirmed-closure`;
- language baseline: `Ordo v0.12 PRE-RELEASE` handoff dated `2026-07-10`.

A0 is an analysis and planning milestone. It does not mutate the APF authoring flow, does not open deferred backlog implementation, and does not change Ordo language/compiler/runtime sources.

## A0 conclusion

APF rc.12 is structurally compatible with Ordo v0.12 as a standard applied module, but it is not yet contract-aligned with the full v0.12 program-level, interaction, process-rail, conversation, prompt-registry, trace-evidence and release-hygiene conventions.

The adaptation should proceed as scoped patches. No APF business-methodology redesign is required at A0.
