# CSG-4 — Authoring-flow, package artifacts, validation, and regression integration

Status: completed.

Conversation Scope Guard is now a conditional generated-playbook capability. APF must capture and confirm the design decision after process-rail and ASR resolution, then generate guard artifacts only when supported. Package release is blocked by state-protection, control-intent, safety, escalation, exit-semantics, trace-hook, or regression failures. APF internal CSG remains disabled.

Regression scenarios cover valid process input, clarification/correction/backtracking, requirement changes, pause/resume/exit, unrelated and repeated deviations, unclassifiable input, safety bypass, state/path preservation, and escalation-counter reset.
