# CSG-0 — Scope and Source-Contract Alignment

Status: completed

APF now recognizes `ORDO-CAP-CSG-001` as the normative source contract for a future optional generated-playbook capability. The APF internal process remains disabled for CSG.

## Fixed decisions

- capability is explicit and optional;
- APF internal default is disabled;
- generated playbooks receive a separate design decision before package assembly;
- canonical modes, classifications, state-protection boundaries, escalation semantics, and trace expectations are adopted without renaming;
- runtime behavior remains outside APF; APF generates contracts, templates, tests, and validation evidence.
