# Missing or unavailable inherited artifacts

## Confirmed unavailable in the current working environment

### `reports/cli_evidence/`

The rc.7 transfer documents describe `reports/cli_evidence/` and several result JSON files as the standard location/model for external CLI evidence. However, this directory and its original evidence files are not present in any currently available rc.7 archive:

- `APF_TRANSFER_PACKAGE_CURRENT_STATE_RC7_CLI_INTEGRATION_CONTRACTS.zip`
- `APF_TRANSFER_PACKAGE_CURRENT_STATE_RC7_CONFIRMED_CLI_INTEGRATION_CONTRACTS.zip`
- `APF_TRANSFER_PACKAGE_CURRENT_STATE_RC7_CLOSURE.zip`

Therefore this transfer package does **not** fabricate or reconstruct those evidence records.

The rc.7 policy and decision state remain preserved through the available rc.7/rc.8/rc.9 reports and confirmation documents. Any future CLI evidence must be generated by the real external tooling and stored under `reports/cli_evidence/` according to the accepted contract.
