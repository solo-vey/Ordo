# Evidence-storage validation tool

This directory contains the validator for the repository's evidence-storage
strategy and binary inventory.

- [`validate_evidence_storage.py`](validate_evidence_storage.py) — checks the
  declared storage strategy against tracked evidence inventory.

For the current evidence strategy, see
[`../../manifests/EVIDENCE_STORAGE_STRATEGY.json`](../../manifests/EVIDENCE_STORAGE_STRATEGY.json).
