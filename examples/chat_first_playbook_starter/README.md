# Chat-first playbook starter example

This checked-in ZIP is a small scenario example. The canonical user download is
the versioned GitHub Release asset identified by
[`../../manifests/ARF_PLAYBOOK_KIT_CURRENT.json`](../../manifests/ARF_PLAYBOOK_KIT_CURRENT.json).

Its contents are built deterministically from [`source/`](source/) by
[`../../tools/build_chat_first_starter.py`](../../tools/build_chat_first_starter.py).

- [`ORDO_CHAT_FIRST_PLAYBOOK_STARTER.zip`](ORDO_CHAT_FIRST_PLAYBOOK_STARTER.zip)
- [`manifest.json`](manifest.json)

Users do not need Python. Maintainers use the builder only to verify that the archive exactly matches its reviewed source files.
