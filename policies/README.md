# Repository policies

This directory contains machine-readable policies and their migration baselines.
The canonical explanatory policy documents are under
[`../docs/policies/README.md`](../docs/policies/README.md).

- `english_only_policy.yaml` — migration-aware English-only repository policy.
- `english_only_migration_baseline.json` — accepted legacy violation baseline.
- `m81_generated_artifacts_policy.yaml` — generated-artifact isolation policy.

Use [`../repo_hygiene.yml`](../repo_hygiene.yml) for repository hygiene categories.
