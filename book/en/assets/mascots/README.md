# Nebu mascot icons

Набір іконок маскота **Nebu** для розділів книги.

## Структура

- `original/` — оригінальні зображення.
- `256x256/` — версії 256×256.
- `128x128/` — версії 128×128.
- `64x64/` — версії 64×64.

## Емоції / сценарії

- `Nebu_base` — базовий маскот.
- `Nebu_attention` — зверніть увагу.
- `Nebu_thinking` — потрібно подумати.
- `Nebu_interesting` — це цікаво.
- `Nebu_idea` — ідея / підказка.
