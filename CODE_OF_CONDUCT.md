# Code of Conduct

## Commitment

The Ordo project is committed to a professional, respectful, and technically rigorous environment.

## Expected Behavior

Participants must:

- communicate respectfully;
- critique ideas and evidence rather than people;
- distinguish verified facts from assumptions;
- disclose relevant conflicts and project relationships;
- respect privacy and confidentiality;
- avoid overstating benchmark, adoption, independence, or release claims;
- support reproducible technical discussion.

## Unacceptable Behavior

Unacceptable behavior includes harassment, threats, discriminatory language, personal attacks, evidence manipulation, concealment of provenance, unauthorized publication of personal or confidential information, malicious code, credentials, and deliberate attempts to bypass release or human-authority controls.

## Technical Integrity

Participants must not:

- label a candidate as canonical without required evidence;
- present model self-evaluation as independent evaluation;
- hide known failures or stale reports;
- alter immutable evidence without a transformation record;
- claim an unverified independence level.

## Scope

This Code applies in official project spaces and when representing the project publicly.

## Reporting

Do not report sensitive conduct concerns in public issues, discussions, pull requests, review comments, or social media.

Use GitHub private vulnerability reporting as the currently approved private repository channel and begin the report title with `Conduct concern:`. See [`SUPPORT.md`](SUPPORT.md#conduct-concerns).

Include only the information needed to assess the concern. Do not disclose unrelated personal information.

## Enforcement

Maintainers may request correction, remove content, close or lock discussions, reject contributions, issue warnings, or restrict participation.

Enforcement decisions should consider context, severity, impact, prior behavior, good-faith correction, and the safety of affected participants.

Retaliation against good-faith reporters is prohibited. Knowingly false reports, evidence fabrication, or use of the reporting process for harassment may result in enforcement action.
