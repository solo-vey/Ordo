# Benchmarks

This directory holds benchmark definitions, runner tooling, scoring contracts, and
supporting data. It does not contain the canonical empirical-evidence record;
see [`../empirical_evidence/README.md`](../empirical_evidence/README.md) for that contour.

- [`ab/README.md`](ab/README.md) — A/B execution and scoring harnesses.
- [`datasets/README.md`](datasets/README.md) — benchmark task datasets.
- [`examples/README.md`](examples/README.md) — example benchmark evidence.
- [`schemas/README.md`](schemas/README.md) — JSON schemas for benchmark records.
- [`taxonomy/README.md`](taxonomy/README.md) — benchmark classifications and aliases.
