# Scientific Question and Hypotheses

## Primary scientific question

For which task classes is Ordo measurably better than an instruction-only approach?

## Main hypothesis

Ordo should provide the largest measurable advantage when successful execution depends on explicit state, deterministic control flow, branching, loops, rollback, validation gates, and safe incomplete termination.

## Null hypothesis

After controlling for model, inputs, task definition, and evaluation method, Ordo provides no material quality advantage over instruction-only execution.

## Evidence rule

Both positive and negative results are retained when the test is valid. The project does not intentionally weaken the instruction-only baseline. A superiority claim is made only for scenarios where the measured evidence supports it.
