# Language Compliance Audit

Audit date: 2026-07-15

## Policy

All package-native text documentation and metadata must be English. Raw execution ZIP archives are immutable evidence and remain byte-for-byte original; they may contain Ukrainian execution content.

## Interpretation

The English-only requirement is satisfied for the package documentation layer. Raw evidence archives are explicitly exempt because translating or editing them would destroy evidentiary integrity. Every admitted raw archive is accompanied by English metadata or an English comparative report.
