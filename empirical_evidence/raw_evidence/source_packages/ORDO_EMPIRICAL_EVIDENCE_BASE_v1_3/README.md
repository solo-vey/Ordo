# Ordo Empirical Evidence Base v1.3

This package is the persistent evidence base for the project’s primary scientific question:

> For which task classes is Ordo measurably better than an instruction-only model?

## Language policy

All package-native documentation, metadata, scorecards, and reports are maintained in English. Immutable raw execution archives are preserved byte-for-byte as source evidence and may contain the language used during execution. Their contents are not treated as package-native documentation and are summarized by English metadata and reports.
