# Submission Context

## Relationship between parties

- **Recipient:** Ordo Language / ARF framework authors and maintainers.
- **Submitter:** an independent external developer team using the released Ordo/ARF package.
- **Nature of work:** the submitter independently created and tested a new playbook using the recipient's language and framework.
- **Framework-author involvement during implementation:** none required for the documented development and pilot cycle.

## Evidence claim

The package supports the following bounded claim:

> A developer team outside the Ordo/ARF implementation team was able to take the released language and framework package, create a new domain playbook, validate and version it, execute a pilot, correct defects, and produce a release candidate without direct participation from the framework authors.

The package does not claim that every external developer will succeed without onboarding, nor that the release candidate is a canonical production release.
