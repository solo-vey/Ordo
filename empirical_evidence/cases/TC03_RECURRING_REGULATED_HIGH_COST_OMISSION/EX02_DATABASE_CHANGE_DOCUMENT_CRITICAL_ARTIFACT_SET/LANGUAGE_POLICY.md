# Evidence Language Policy

All evidence-base governance, inventories, manifests, status records, integrity reports, evaluation analyses, and comparative scorecards created or updated for this test case are maintained in English.

Immutable source inputs, model-visible analyst corpora, templates, launch prompts, and original returned execution artifacts are preserved byte-for-byte in their original language. They are evidence objects and are not translated because translation would alter the tested input or returned evidence.
