# BL-ORDO-018 Benchmark Evidence

This directory contains the immutable raw evidence for the closed
BL-ORDO-018 blind model-comparison benchmark.

## Retained assets

- `MODEL_1_API_RESULTS.zip` — API responses for the first compared model.
- `MODEL_2_API_RESULTS.zip` — API responses for the second compared model.
- `BLIND_SCORING_RESULTS.zip` — blinded scoring records, pair results, and
  validation summary derived from the two result sets.

The archives are preserved byte-for-byte. Their repository SHA-256 values are
recorded in the root `SHA256SUMS.txt`; benchmark closure context remains in
`reports/m87_external_evidence/M87_5_BL_ORDO_018_FINAL_CLOSURE_REPORT.json`.

This is a benchmark-specific evidence record. It is distinct from the current
task-case evidence under the rest of `empirical_evidence/`.
