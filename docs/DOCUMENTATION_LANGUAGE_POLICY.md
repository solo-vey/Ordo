# Documentation Language Policy

English is the only authoring language for repository documentation, specifications, release notes, reports, examples, public posts, and new textual artifacts.

The only maintained non-English documentation tree is the localized book under `book/uk/`. Translations must be derived from canonical English sources and tracked through explicit version and hash metadata.

Existing non-English repository files are migration debt. They must not be used as templates for new documentation and should be translated or retired in controlled batches.
