# Publication Decision Matrix

| Option | Status for v0.12.0-preview | Reason |
|---|---|---|
| Private/internal only | available but not selected | Too restrictive for review handoff. |
| Source-available preview | selected | Allows controlled review/public preview without granting OSS rights. |
| Open-source release | deferred | Requires explicit owner/legal license choice. |
| Commercial/proprietary release | out of scope | Not the goal of this milestone. |

## Selected decision

`source-available preview candidate` is selected for M39.

## Deferred decision

Final open-source licensing remains deferred. It is no longer a blocker for preview publication, but it remains a blocker for calling the repository an open-source release.
