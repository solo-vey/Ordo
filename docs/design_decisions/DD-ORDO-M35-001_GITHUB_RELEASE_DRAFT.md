# DD-ORDO-M35-001 — GitHub Release Draft

## Status

Accepted

## Decision

M35 prepares a GitHub release draft for Ordo v0.12 Public MVP Preview without changing the publication mode.

The repository remains a source-available preview candidate until the project owner selects the final license.

## Reason

M34 prepared the workspace for external review. The next practical step is to make the GitHub release story explicit: tag, release title, reviewer path, checklist and publication constraints.

## Impact

- Adds `docs/github_release/`.
- Adds a release draft and release checklist.
- Keeps the license decision open.
- Does not claim open-source status.

## Validation expectation

M35 is valid if the release draft is clear, the license blocker is explicit, and the root README/CHANGELOG point to the new release preparation layer.
