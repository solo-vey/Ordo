# Publication Decision — Ordo v0.12 Public MVP

## Decision

Ordo v0.12 may be published as a **public source-available preview candidate**, but not yet as an open-source release.

## Reason

The Process Rail model, Project Builder, Hybrid Execution model, CLI helper layer, reference packages and registry artifacts are coherent enough for external review. However, a final open-source license has not been selected.

## Publication mode

Allowed current mode:

```text
public GitHub repository / source-available preview / external review candidate
```

Not allowed until a license is selected:

```text
open-source release / production reuse grant / package redistribution grant
```

## Human-facing wording

Use:

```text
Ordo v0.12 Public MVP candidate is available for review as a source-available preview. The open-source license is pending.
```

Avoid:

```text
Ordo is open source.
```

## Follow-up decision

A project owner must choose the license strategy before the repository is announced as open-source.
