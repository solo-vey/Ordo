# M48 Final Handoff Package

M48 is the reviewer-facing handoff layer for `v0.12.0-preview-rc1`.

It adds no new Ordo language semantics, CLI behavior, package business logic, or book PDF output. It collects the release-candidate state into a small review route so an external reviewer can inspect the package without reconstructing the milestone history.

## Start here

1. Read `REVIEWER_START_HERE.md`.
2. Read `M48_FINAL_HANDOFF.md`.
3. Use `REVIEW_QUESTIONS.md` to record external feedback.
4. Run the checks listed in the handoff if you want to verify the package locally.

## Scope

M48 covers:

- source workspace archive;
- Developer bundle archive;
- release-candidate freeze evidence;
- external audit checklist and self-audit evidence;
- go/no-go validation layer;
- known limitations and non-goals.
