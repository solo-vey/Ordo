# Ordo v0.12.0-preview — Final Public Preview Release Notes

## Summary

This preview introduces Ordo as a Process-Rail-centered AI process language.

The release demonstrates two main flows:

1. **AI-guided authoring** — a PM/analyst works with AI Ordo Developer to create or modify Ordo projects.
2. **Hybrid execution** — AI Ordo Executor runs a compiled Semantic JSON IR while deterministic CLI helpers stabilize state, gates and next-step checks.

## Important status note

This is a source-available public preview candidate, not an open-source release.
