# M40 — Final Public Preview Release Package

## Status

`v0.12.0-preview` is packaged as a final **source-available public preview** release candidate.

This is not an open-source release. The open-source license decision remains deferred by design, but it is no longer a blocker for source-available preview publication.

## What is included

- Ordo language/spec v0.12 with Process Rail as the core model.
- Semantic JSON IR model with Process Rail, conversation semantics and helper-tool hooks.
- `ordo-cli` v0.22.0 with deterministic helper commands.
- Reference packages:
  - `ordo_project_builder`
  - `ordo_hybrid_executor`
  - `history_event_guided_intake`
- Book/spec/docs aligned to the Process Rail model.
- GitHub release draft and review handoff materials.
- Publication manifests and validation reports.

## Publication mode

The package may be published as a source-available preview if the repository description, README and release notes do not call it open-source.

## Release blockers

No blocker remains for source-available preview publication.

Open-source publication remains blocked until a final OSI-compatible license is selected.
