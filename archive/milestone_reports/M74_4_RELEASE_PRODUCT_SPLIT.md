# M74.4 — Release Product Split

Defines three independent release products:

1. `language_baseline` — specification, registries, schemas, CLI/toolchain, tests, utilities, examples/reference packages, developer docs.
2. `process_archive` — milestone and closure evidence, historical reports, benchmark outputs.
3. `book_source` — Ukrainian and English book sources and assets.

The split is generated as directories for validation. No intermediate ZIP archives are produced.
