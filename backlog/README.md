# Backlog and maturity state

This directory contains the current human-readable planning and maturity documents.

- [`CONSOLIDATED_BACKLOG.md`](CONSOLIDATED_BACKLOG.md) is the canonical Markdown backlog synchronized with `manifests/CONSOLIDATED_BACKLOG.json`.
- [`CURRENT_MATURITY_STATE.md`](CURRENT_MATURITY_STATE.md) is the concise human-readable maturity summary.
- [`FUTURE_BACKLOG.md`](FUTURE_BACKLOG.md) preserves deferred historical and future-work notes.

Machine-readable status remains under `manifests/`. Historical snapshots under `archive/`, `reports/`, and `TRANSFER_*` remain provenance rather than present-tense status.
